# Node Boilerplate

Generate a new project based on [NodeJS](http://nodejs.org/).

## Requires

- [Node.JS](http://nodejs.org/)

## Instructions

1. Clone the project

  ```bash
  git clone git://github.com/vitorbritto/boilerplates
  ```
2. Choose **CLI Tool**

3. Profit! :)
