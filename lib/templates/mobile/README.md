# Mobile Boilerplate

Generate a new project based on [Phonegap]().

## Requires

- [Node.JS](http://nodejs.org/)
- [Bower](http://bower.io)
- [Grunt](http://gruntjs.com)


## Instructions

1. Clone the project

  ```bash
  git clone git://github.com/vitorbritto/boilerplates
  ```
2. Choose **Benm Stack**

3. Profit! :)

- [PhoneGap](http://phonegap.com/)
- [Zepto.js](http://zeptojs.com)
