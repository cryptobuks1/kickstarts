# General Boilerplate

Generate a new project based on General Scaffolding.

- [Node.JS](http://nodejs.org/)
- [Bower](http://bower.io)
- [Grunt](http://gruntjs.com)
- [Ruby](https://www.ruby-lang.org/pt/)
- [Sass](http://sass-lang.com/)
- [Compass](http://compass-style.org/)
